# 🎬 ALURIX - Netflix Clone

Projeto desenvolvido com o objetivo de replicar a interface da Netflix, aplicando tecnologias fundamentais do desenvolvimento web e integrando o uso de Inteligência Artificial no fluxo de trabalho.

---

## 🚀 Funcionalidades

* **Seleção de Perfis:** Interface interativa para escolha de usuário.
* **Catálogo:** Exibição de filmes e séries com layout inspirado na plataforma original.
* **Responsividade:** Design adaptável para diferentes tamanhos de tela (Desktop, Tablet e Mobile).
* **Interatividade:** Manipulação do DOM para alternância de temas e estados.

## 🛠️ Tecnologias e Ferramentas

O projeto foi construído utilizando:

* **Linguagens:** HTML5 (Semântico), CSS3 e JavaScript (Vanilla).
* **Ambiente de Desenvolvimento:** VSCode.
* **IA como Copiloto:**
    * **GitHub Copilot:** Auxílio na geração de ideias e esclarecimento de dúvidas técnicas.
    * **Gemini:** Utilizado para geração de imagens de perfil, estruturação lógica e correção da semântica do código.
